project5：Impl Merkle Tree following RFC6962

项目说明：

默克尔树（Merkle Tree）是一种基于哈希的数据结构，它是哈希列表的一种推广。它是一种树形结构，其中每个叶子节点是一个数据块的哈希值，每个非叶子节点是其子节点的哈希值的哈希。通常，默克尔树的分支因子为2。

默克尔树的构建过程如下：

1.首先，将要存储或者验证的数据分割成固定大小的数据块，并对每个数据块计算一个哈希值。这些哈希值就是默克尔树的叶子节点。

2.然后，将相邻的1两个叶子节点的哈希值连接起来，并对这个连接后的字符串再次计算一个哈希值。这个哈希值就是两个叶子节点的父节点。

3.重复上述步骤，直到只剩下一个节点为止。这个节点就是默克尔树的根节点，也叫做默克尔根（Merkle Root）。

4.如果在某一层中，节点的数量是奇数，那么就将最后一个节点复制一份，并与自己连接起来，再计算一个哈希值作为父节点。
