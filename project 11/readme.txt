project11：impl sm2 with RFC6979

项目说明：

SM2是中国国家密码管理局于2010年提出的一种椭圆曲线公钥密码算法（Elliptic Curve Public Key Cryptography，简称ECC），属于非对称密码算法的一种。它是中国自主设计的密码算法，并在2012年被正式发布。

SM2算法基于椭圆曲线离散对数难题，使用了中国政府发布的一条特定的椭圆曲线参数作为公开参数。它具有以下特性：

1. 安全性高：SM2使用了256位的椭圆曲线，提供了较高的安全性，能够有效抵御常见的密码学攻击。

2. 效率高：相较于传统的RSA算法，SM2在相同的安全性水平下，具有更高的计算效率和更小的密钥和签名长度。

3. 支持数字签名和密钥交换：SM2可以用于数字签名、密钥交换和公钥加密等场景，满足不同应用的安全需求。

RFC6979是一种确定性非ceslping随机数生成算法。结合SM2和RFC6979可以实现更安全和高效的数字签名算法。
